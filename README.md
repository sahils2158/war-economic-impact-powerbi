# 🌍 War Economic & Livelihood Impact — Power BI Dashboard

## 📊 Project Overview
An interactive Power BI dashboard analyzing the economic devastation caused by major wars 
(WWII to Israel-Iran Conflict 2025-26) across 100,000 synthetic records.

## 🔍 Key Insights
- Wars cause an average GDP drop of ~31.6%
- Poverty rates increase by an average of X% during conflicts
- Reconstruction costs are 2-3x higher than war costs
- Middle East and Africa show highest food insecurity rates

## 🛠️ Tools Used
- Python (Pandas) — Data Cleaning & Feature Engineering
- Microsoft Power BI Desktop — Dashboard Development
- DAX — Custom Measures & KPIs
- Power Query — Data Transformation

2. **Human Cost** — Poverty, Unemployment, Food Insecurity

4. **Timeline & Regions** — Historical trends, Geographic map
